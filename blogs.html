• Moments and Deviations is a blog devoted to fun and interesting problems in Computer Science and Mathematics for the purpose of keeping me continually learning new ideas and practicing existing skills. If you have interesting problems, feel free to send those to me and I shall post them (with perhaps a solution).

    Currently, the blog is stalled while I spend my free time working on my go projects, but I hope to start it up again at some point.

  • The Hamete Go Group is a blog devoted to studying the game go. I created this blog so I could still be a part of the awesome Portland Go community despite having moved away.

    I really like having a Go blog, but after spending some time programming and also exploring existing client-side Go programs, I think EidoGo would be more ideal than GoSWF, which is what I used for the Hamete Go Group @ Blogspot. Unfortunately, this would require me to host the blog on my website, which will take a bit of work. Like Moments and Deviations, I am planning on picking this project up again when I get some more time.
